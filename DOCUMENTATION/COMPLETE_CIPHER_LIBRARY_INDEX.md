## CIPHER PARAMETER TYPES

### Parameter Categories

#### No Parameters
Most ciphers work without parameters (standard transformation):
- Atbash, ROT13, ROT47, Reverse, Mirror, etc.

#### Text Parameters (Key-based)
Require a text key for transformation:
- **Key** - Vigenère, Beaufort, Porta, Playfair, Four-Square, etc.
- **Key1, Key2** - Columnar Double, Four-Square variants

#### Numeric Parameters
Require numeric configuration:
- **Shift** - Caesar, rotation ciphers (1-25)
- **Shift1, Shift2** - Alternating Shift (dual shifts)
- **Rails** - Rail Fence, Zigzag (2-10)
- **Size** - Block Reverse (block size)
- **Mod** - Modular Cipher (modulus)
- **Mult** - Multiplicative (multiplier)
- **Exp** - Exponential (exponent)

#### Optional Parameters
Some ciphers accept optional parameters:
- Most work without parameters but accept them for customization

## API USAGE EXAMPLES

### Get All Ciphers
```
GET /api/ciphers
```
Returns list of all 142 ciphers with metadata.

### Encrypt Text
```
POST /api/encrypt
Content-Type: application/json

{
  "cipher": "caesar",
  "text": "Hello World",
  "params": {"shift": 3}
}
```

### Decrypt Text
```
POST /api/decrypt
Content-Type: application/json

{
  "cipher": "caesar",
  "text": "Khoor Zruog",
  "params": {"shift": 3}
}
```

### With Key-Based Cipher
```
POST /api/encrypt
Content-Type: application/json

{
  "cipher": "vigenere",
  "text": "Hello World",
  "params": {"key": "SECRET"}
}
```
